# Praca roczna — EFX for 4 agents and 8 goods

Build the PDF with:

```sh
make        # runs pdflatex + biber + pdflatex x2, produces efx.pdf
make clean  # remove build artifacts
```

Requires a TeX distribution with `pdflatex` and `biber` (e.g. TeX Live).

- `efx.tex` — the report.
- `efx.bib` — bibliography.
